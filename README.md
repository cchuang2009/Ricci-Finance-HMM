# Ricci Finance v12

Ricci Finance v12 combines rolling financial networks, Ollivier–Ricci curvature,
capital-flow weighting, Hidden Markov regimes, and synchronized 3-D market-story
animation.

## Quick start

```bash
uv venv
source .venv/bin/activate
uv sync
uv run streamlit run app_v12.py
```

Alternatively:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app_v12.py
```

## Current release

This package is v12 Phase 3, the **HMM Story Manifold** release. It adds HMM
posterior probabilities, frame-to-frame narratives, synchronized 3-D animation,
capital-flow interpretation, and selected-frame diagnostics.

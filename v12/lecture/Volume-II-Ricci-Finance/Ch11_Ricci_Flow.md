# Chapter 11 — Ricci Flow Projection

# 第十一章：Ricci Flow 投影

---

## Learning Objectives / 學習目標

### English

- Compare observed geometry with flowed geometry.

### 繁體中文

- 比較觀測幾何與 Flow 後幾何。

## Purpose / 目的

### English

Ricci Flow smooths geometry and highlights persistent structures.

### 繁體中文

Ricci Flow 平滑幾何並凸顯持續性結構。

## Limitations / 限制

### English

Flow is a structural transformation, not a direct future-price forecast.

### 繁體中文

Flow 是結構轉換，不是直接的未來價格預測。

## Functions Used / 使用函式

### `run_ricci_flow()`

**Purpose:** Execute flow iterations.

**用途：** 執行 Ricci Flow 迭代。

**Inputs / 輸入：** curved graph, iterations

**Outputs / 輸出：** flowed graph

### `flow_delta_features()`

**Purpose:** Measure before-after changes.

**用途：** 衡量 Flow 前後變化。

**Inputs / 輸入：** original and flowed graphs

**Outputs / 輸出：** delta features

This chapter connects the mathematical idea, the financial interpretation, and the software implementation used in Ricci-Finance v12.

本章將數學概念、金融意義與 Ricci-Finance v12 的程式實作連結在一起。

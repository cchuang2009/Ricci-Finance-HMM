# Introduction

Ricci Finance v14 studies a market as a rolling weighted graph. Tickers are nodes;
return relationships create edges; dollar volume supplies capital mass; and
Ollivier–Ricci curvature measures local geometric cohesion or fragility.

The final version unifies data, graph geometry, regime inference, validation and
3-D storytelling under one reproducible yfinance workflow.

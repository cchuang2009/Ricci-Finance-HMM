# Research Limits and Extensions

v14 is the end version of this implementation line, not a claim that the open
research problem is solved.

Important limits include survivorship bias, yfinance data quality, state-label
non-identifiability, multiple-testing risk and sensitivity to graph parameters.

Possible separate research branches include switching state-space models,
particle filters, graph neural networks, persistent homology and calibrated
probabilistic graph forecasting. These should be new projects rather than
silently altering the v14 baseline.

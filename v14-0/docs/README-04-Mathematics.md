# Mathematics

For rolling log returns \(r_i(t)\), pairwise correlation is converted into the
standard correlation distance

\[
d_{ij}=\sqrt{2(1-\rho_{ij})}.
\]

Capital similarity can contract this raw distance without changing the measured
correlation. Ollivier–Ricci curvature compares transport between neighboring
probability measures. Negative curvature is interpreted as bridge-like or
fragile geometry; positive curvature as locally cohesive geometry.

The HMM estimates hidden regimes from rolling graph observables. Viterbi chooses
the most probable complete state path, while posterior decoding chooses the most
probable state independently at each frame. HSMM adds explicit duration.

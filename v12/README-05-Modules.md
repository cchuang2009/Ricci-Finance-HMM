# Modules

| Module | Responsibility |
|---|---|
| `data.py` | Market download, synthetic data, validation, returns |
| `features.py` | Volatility, momentum, drawdown, volume features |
| `graph.py` | Correlation distances, graph construction, components |
| `capital.py` | Node capital mass and edge capital transport |
| `ricci.py` | Curvature and Ricci-flow projection |
| `rolling.py` | Rolling frames and graph statistics |
| `regimes.py` | HMM fitting, states, and posterior probabilities |
| `story.py` | Frame comparison and evidence-based narrative |
| `layouts.py` | Stable layouts across rolling frames |
| `visualization.py` | Static and animated 3-D manifolds |
| `plots.py` | Diagnostic and regime charts |
| `surgery.py` | Separation-risk ranking and optional experiments |
| `diagnostics.py` | Edge and graph inspection tables |

# Mathematics

The base financial distance for correlation \(ho_{ij}\) is

\[
d_{ij}=\sqrt{2(1-ho_{ij})}.
\]

Capital similarity can contract this distance to create an effective graph
metric. Ollivier–Ricci curvature is then computed on the weighted graph.

For every rolling frame, v12 records statistics including mean curvature,
curvature dispersion, negative-edge ratio, graph density, component structure,
edge stability, capital concentration, and total edge capital flow.

The HMM observes the sequence of frame-level feature vectors and estimates a
latent state plus posterior probabilities. The state is descriptive rather than
a guaranteed forecast of future returns.

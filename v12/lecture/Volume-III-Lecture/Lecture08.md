## Functions Used / 使用函式

### `render_frame()`

**Purpose:** Draws one frame.

**用途：** 繪製單一畫格。

**Inputs / 輸入：** lecture data and configuration

**Outputs / 輸出：** result used in the lecture

### `assemble_animation()`

**Purpose:** Creates GIF or MP4.

**用途：** 建立 GIF 或 MP4。

**Inputs / 輸入：** lecture data and configuration

**Outputs / 輸出：** result used in the lecture

### `render_animation_tab()`

**Purpose:** Displays animation in Streamlit.

**用途：** 在 Streamlit 顯示動畫。

**Inputs / 輸入：** lecture data and configuration

**Outputs / 輸出：** result used in the lecture
